# Bulk CRUD Spring Boot Project

## Overview
This is a Spring Boot project for managing **Records** with bulk CRUD operations. The project allows you to:

- Create multiple records at once (`POST /records/bulk`)
- Retrieve all records (`GET /records`)
- Input validation (name, email, age)
- Prevent duplicacy
- Show proper Https codes
- Proper exception handling
- Unit tests for service and controller layers

---

## Tech Stack
- **Backend:** Spring Boot 3
- **Database:** MySQL 8
- **ORM:** Spring Data JPA (Hibernate)
- **Testing:** JUnit 5, Mockito
- **Build Tool:** Maven
- **Java Version:** 17

---
