# Bulk CRUD Spring Boot Application


---

## Features

- **CRUD Operations**: Create, Read, Update, Delete single or bulk records.
- **Bulk Import**: Support for JSON and Excel file upload.
- **Exception Handling**: Global exception handler for validation and runtime errors.
- **Pagination**: Fetch records in pages using REST endpoints.

---

## Technology Stack

- Java 17
- Spring Boot 3.4.12
- Spring Data JPA
- H2 / MySQL Database
- Maven
- RESTful APIs
- Excel processing (`Apache POI`)
